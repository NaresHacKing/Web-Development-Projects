Anchor Tag Attributes:

  1. To download anything with this attribute
  2. href The URL that the hyperlink points to
    1. Google
    2. Github
  3. target Where to display the linked url (a tab, new window, or <iframe>)
    1. _self: Dispay the content in the current browising context
    2. _blank: Opens a new browser window/tab and load the page.
    3. _parent: Loads the parent document into an iframe element inside the current frameset
    4. _top: The top most browing context
